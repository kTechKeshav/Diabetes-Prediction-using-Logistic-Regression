Diabetes Prediction Project

This project uses a machine learning model (Logistic Regression) to predict whether a patient is diabetic or not based on medical parameters.

  • Input features: Pregnancies, Glucose, Blood Pressure, Skin Thickness, Insulin, BMI, Diabetes Pedigree Function, Age
  • All data is processed and standardized for accurate prediction
  • Model is trained and tuned for best results
